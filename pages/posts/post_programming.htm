Mijn programmeer biografie

Sinds 2018 ben ik lekker bezig geweest met programmeren, ik begon met het maken van kleine programma's in de programmeertaal BASIC in een app genaamd "Lowres Coder". Daarna maakte ik automatische berekeningen zoals de abc formule en later spelletjes zoals Minesweeper op mijn rekenmachine in TI BASIC. Daarna begon ik met spelletjes en functionele programmas te maken in BASIC in Lowres Coder en later in zijn opvolger LowresNX (en ik ben nog steeds niet gestopt). Vervolgens leerde ik C met Arduino om voor mijn vwo profielwerkstuk een robot te maken. In in 2022 leerde ik Processing JavaScript en Processing Java om een simulatie te maken voor de Trashbot van de engineering opleiding van de HZ die ik toen volgde. In 2023 leerde ik AutoHotKey omdat het gewoon handig is. Ook leerde ik React en React Native zodat ik een idee van een app kon ontwikkelen en Unity om een ander app idee te ontwikkelen. Het jaar erna in 2023-2024 leerde ik alle ins en outs van C aan de Technische Universiteit van Eindhoven. Ik haalde C wel maar helaas niet mijn BSA. Daarna heb ik nog met C++ gewerkt, nog meer met Processing Java en met HTML met JavaScript (ik waardeer de simpliciteit en veelzijdigheid van deze twee).
